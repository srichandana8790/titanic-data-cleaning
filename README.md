# Titanic Data Cleaning Project

## Overview
Cleaned the Titanic dataset using Python and Pandas as part of my Data Analytics portfolio.

## Steps Performed
- **Missing Values** - Filled Age with median, Embarked with mode, dropped Cabin
- **Inconsistent Data** - Checked Sex and Embarked columns for inconsistencies
- **Scaling** - Standardized Age, Normalized Fare using Scikit-learn
- **Encoding** - Label Encoded Sex, One Hot Encoded Embarked

## Tools Used
- Python
- Pandas
- Scikit-learn
- Kaggle Notebooks

## Dataset
Titanic - Machine Learning from Disaster (Kaggle)

## Output
Cleaned dataset saved as `titanic_cleaned.csv`
